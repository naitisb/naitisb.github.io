# naitisb.github.io

Personal/professional website for Naiti S. Bhatt, built as a plain static HTML/CSS/JS site (no build step, no Jekyll). Pages: Home, Portfolio, Presentations, CV, Photos, Contact.

Served directly via GitHub Pages from the `master` branch root. To preview locally, serve the directory with any static file server (e.g. `python3 -m http.server`) — the pages also work by opening the `.html` files directly in a browser.
